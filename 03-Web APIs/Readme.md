## Web APIs

Web APIs is an api exposed by a web component , allowing other components to interact with it

#### Common characteristics of Web APIs 

- Platform agnostic 
- Standard Protocols over HTTP
- Usually Request/Response Based

#### Differentiated by 

- Request format 
- Request Contents 
- Response Format
- Response Contents

#### Types of Web APIs 

- SOAP
- REST
- GraphQL
- GRPC
